GitHub 经典项目榜：8 个值得长期收藏的 AI 基础设施

这一组不是追新，而是适合补课。

如果你想理解 AI 应用、Agent、本地模型、提示词、网页抓取这些方向，很多绕不开的项目都在这里。

收藏 = GitHub Star
热度 = 近 17 天新增 Star
链接可以复制到浏览器打开。

01｜hermes-agent
一句话看懂：普通 AI 助手缺少长期适应和个性化成长能力，它想做一个会随你成长的 Agent。
收藏：198,805 Star｜热度：+17,565
链接：https://github.com/NousResearch/hermes-agent

02｜transformers
一句话看懂：开发者不想从零实现模型加载、训练和推理，就会用到它。
收藏：161,771 Star｜热度：暂无新增数据
链接：https://github.com/huggingface/transformers

03｜ollama
一句话看懂：想在本地跑开源模型，但不想折腾复杂部署。
收藏：174,650 Star｜热度：+1,448
链接：https://github.com/ollama/ollama

04｜prompts.chat
一句话看懂：不会写 prompt 时，可以先看社区里的成熟模板。
收藏：164,019 Star｜热度：暂无新增数据
链接：https://github.com/f/prompts.chat

05｜langchain
一句话看懂：从模型调用到工具编排、链路追踪都自己写太麻烦时，用它搭 LLM 应用。
收藏：139,811 Star｜热度：暂无新增数据
链接：https://github.com/langchain-ai/langchain

06｜open-webui
一句话看懂：本地模型或 API 有了，但缺一个顺手的聊天和管理界面。
收藏：142,518 Star｜热度：暂无新增数据
链接：https://github.com/open-webui/open-webui

07｜firecrawl
一句话看懂：给 AI 喂网页内容时，自己写爬虫太不稳定。
收藏：136,160 Star｜热度：暂无新增数据
链接：https://github.com/firecrawl/firecrawl

08｜GitHubDaily
一句话看懂：不知道每天该看哪些有趣实用的 GitHub 项目，可以长期关注它。
收藏：46,839 Star｜热度：+153
链接：https://github.com/GitHubDaily/GitHubDaily

我自己的感觉是：
这一组适合当 AI 工具生态的“底层地图”来看。

本地模型、模型框架、提示词、Agent 工程、聊天界面、网页抓取、项目发现，都是长期会反复用到的能力。

如果只挑 3 个先看，我会看：
ollama、open-webui、firecrawl。

一个负责本地模型，一个负责界面，一个负责把网页内容变成 AI 能用的数据。
