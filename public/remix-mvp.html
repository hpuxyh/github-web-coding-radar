别人已经做出来的例子

为什么做

因为很多人想做网页，但不想先被代码拦住。

GitHub 上有很多已经能打开的好页面。我们先把这些页面整理成可编辑模板，让新手先做出一个能展示的作品，再慢慢理解背后的代码。

需求在变多

越来越多人想做个人主页、小工具、作品集和内容页，但直接看 GitHub 项目太难。

第一步要更轻

新手先需要“能看、能改、能展示”，不是一开始就学命令行、目录结构和部署。

我们能做出来

第一版先选能直接打开的网页，拆成标题、卡片、图片和按钮，让 AI 对话直接改预览。
